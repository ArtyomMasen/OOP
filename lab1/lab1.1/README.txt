How to use:

1. Compile an exe file: dart compile exe copyfile.exe
2. Run program: copyfile.exe <input file name> <output file name>

Case #1: copyfile.exe file1.txt file2.txt
Case #2: copyfile.exe file.txt file2.txt
Case #3: copyfile.exe inputFile.txt outputFile.txt
Case #4: (In this case you created an output file) copyfile.exe input.txt output.txt
